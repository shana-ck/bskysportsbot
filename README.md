# Bsky Sports Bot

This is a framework for running a bot which mirrors posts from Mastodon. It was built primarily using [@skyware/bot](https://github.com/skyware-js/bot) and [masto.js](https://github.com/neet/masto.js/).

## Usage

Please note: The Skyware Bot repo is archived. In order to prevent links from breaking the bot, there is a post-install script to patch the bot. Make sure these changes are applied.
